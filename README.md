# 🌿 BogotáVerde Pro
Aplicación integral de salud y bienestar diseñada para la comunidad de Bogotá (especialmente Kennedy).

## 🚀 Funciones Principales
1. **Calculadora IMC (OMS):** Análisis de peso basado en estándares internacionales.
2. **Recuperación de Sueño:** Recomendaciones de ejercicio según tu descanso.
3. **Sensor de Pasos:** Contador en tiempo real usando el acelerómetro del celular.
4. **Reporte PDF:** Generación de informes de rendimiento cada 15 días.

## 🛠️ Tecnología
Desarrollado en un **Nokia G10** usando **Termux**, **Acode**, HTML5, CSS3 y JavaScript.
